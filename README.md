# Pubs list
## Back end Project setup
### Creates and runs a container with postgres (one time need)
```
docker-compose up -d
```

## Front end Project setup
### /src/main/webapp
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Lints and fixes files
```
npm run lint
```
